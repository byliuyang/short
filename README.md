# Short
URL shortening service

## Preview
![Home](doc/screenshot/home.png)

## Prerequisites
- Yarn v1.17.3
- Go   v1.12.7

## Getting Started

1. Build App
```bash
./bin/build
```

2. Start server
```bash
./build/short start --www build/web
```

3. Visit [http://localhost](http://localhost)
